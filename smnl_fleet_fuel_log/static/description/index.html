SMNL Fleet Tanker Master

Using this module allow user to manage Tanker Master and create vendor bills

  • Features
  • Create Tanker Master
  • Vendor Bill Generated
  • Quantity Consumed in Fuel log

How to Use

  • Fleet -> Configuation -> Tanker Master
  • Created Vendor Bill in Tanker master
  • Auto generated Vendor Bill as Tanker master
  • List view of Tanker master
  • Confirmed Tanker is showing in Fuel Log
  • Consumed Quantity
  • Available Quantity in Tanker Master
