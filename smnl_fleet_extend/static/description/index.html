SMNL Fleet Management

Using this module allow user to manage fleet model & spare-part as product, allow to manage fleet chassis, engine, register/temporary number at purchase or at manual inventory adjustment time, allow manage inventory, purchase, sales, account management, allow searching functionality, allow to print fleet cost & fuel report.

  • Features
  • Create fleet model as product
  • Purchase fleet
  • Fleet Inventory

Configuration

  • Settings -> General Settings -> Inventory

How to Use

  • Create fleet model as product
  • Auto generated Product as fleet model

  • Vendor Store

  • Purchase Quotation/Order

  • Create Purchase Quotation & Confirm order

  • Receive Products

  • Click on button to add Lot/Serial/Chassis Number, Register number, Engine Number

  • If product as fleet model then insert Chassis Number, Register number, Engine Number & it should be unique then click on confirm

  • If product is normal then insert Chassis Lot/Serial Number & it should be unique then click on confirm

  • Click on validate & done the purchase order

  • Auto generated fleet

  • For manual fleet inventory. Go to product which as a fleet model & click update Qty On Hand

  • Click on Start Inventory

  • If product as fleet model then insert Chassis Number, Register number, Engine Number & it should be unique then click on Validate Inventory

  • Auto generate fleet via manual inventory
